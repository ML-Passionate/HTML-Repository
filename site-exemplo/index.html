HTML Commands Repository

This page serves as a repository of common HTML commands, their descriptions, and examples.

Basic Structure

  • <!DOCTYPE html>

    Defines the document type and version of HTML.

    <!DOCTYPE html>
  • <html>

    The root element of an HTML document.

    <html></html>
  • <head>

    Contains meta-information about the document, such as title, styles, and scripts.

    <head><title>Page Title</title></head>
  • <body>

    Contains the content of the document that is visible to the user.

    <body><h1>Hello World</h1></body>

Text Formatting

  • <h1> to <h6>

    Defines headings, with <h1> being the most important and <h6> the least.

    <h1>Heading 1</h1>
    <h2>Heading 2</h2>
  • <p>

    Defines a paragraph.

    <p>This is a paragraph.</p>
  • <strong>

    Defines important text, typically displayed in bold.

    <strong>This text is important.</strong>
  • <em>

    Defines emphasized text, typically displayed in italics.

    <em>This text is emphasized.</em>
  • <br>

    Inserts a single line break.

    This is a line of text.<br>This is another line.

Links and Images

  • <a>

    Defines a hyperlink.

    <a href="https://www.example.com">Visit Example</a>
  • <img>

    Embeds an image in the document.

    <img src="image.jpg" alt="Description of image">

Lists

  • <ul>

    Defines an unordered list.

    <ul>
      <li>Item 1</li>
      <li>Item 2</li>
    </ul>
  • <ol>

    Defines an ordered list.

    <ol>
      <li>First item</li>
      <li>Second item</li>
    </ol>
  • <li>

    Defines a list item.

    <li>List item</li>

Forms

  • <form>

    Defines a form for user input.

    <form action="/submit" method="post">
      <input type="text" name="username">
      <input type="submit" value="Submit">
    </form>
  • <input>

    Defines an input field.

    <input type="text" name="username" placeholder="Enter your username">
  • <button>

    Defines a clickable button.

    <button type="submit">Submit</button>

Tables

  • <table>

    Defines a table.

    <table>
      <tr>
        <th>Header 1</th>
        <th>Header 2</th>
      </tr>
      <tr>
        <td>Data 1</td>
        <td>Data 2</td>
      </tr>
    </table>
  • <tr>

    Defines a table row.

    <tr><td>Data</td></tr>
  • <td>

    Defines a table cell.

    <td>Data</td>
  • <th>

    Defines a table header cell.

    <th>Header</th>

Semantic Elements

  • <header>

    Defines a header for a document or section.

    <header>
      <h1>Website Title</h1>
    </header>
  • <footer>

    Defines a footer for a document or section.

    <footer>
      <p>Copyright 2023</p>
    </footer>
  • <section>

    Defines a section in a document.

    <section>
      <h2>Section Title</h2>
      <p>Section content.</p>
    </section>
  • <article>

    Defines an independent, self-contained content.

    <article>
      <h2>Article Title</h2>
      <p>Article content.</p>
    </article>
  • <nav>

    Defines navigation links.

    <nav>
      <a href="/home">Home</a>
      <a href="/about">About</a>
    </nav>

Other Elements

  • <div>

    Defines a division or section in a document.

    <div>This is a div element.</div>
  • <span>

    Defines a small section of text within a line.

    <p>This is a <span style="color:red;">red</span> word.</p>
  • <iframe>

    Embeds another HTML page within the current page.

    <iframe src="https://www.example.com" width="300" height="200"></iframe>
